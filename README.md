# Ontological Freedom, Non-Being, and Social Engineering

**Independent Research Project**

This project develops an ontological–phenomenological framework
distinguishing between:
- Ontological freedom
- Social freedom
- The social production of non-being

It critically re-examines:
- Slavery and enslavement
- Myth as narrative ontology
- Language as world-constituting
- Responsibility prior to law and ethics
